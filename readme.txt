Сборка проектов разными сборщиками:
Все сборщики поддерживают стнадартные цели: build (install для maven), clean, test

Особенности сборки

Ant

0. Управляем зависимостями использованием ivy, если ivy не установлена - библиотека подкачивается, теребуется повторный запуск задания
1. Цель resolve скачивает необходимые библиотеки
2. Собирает war или jar архив (build-jar и build-war цели соответствено);
3. Артефакты ложатся в папку build\libs
4. Есть цель run, запускающая jar на выполнение (перед запуском выполняет сборку, выводит в консоль "WebApp!!!!")
5. clean-all дополнительно чистит кеш ivy

Maven

1. Стандартные цели
2. Артефакты создаются в папке target

Gradle

1. Артефакты создаются в папке  build\libs
2. Определены 2 дополнительных задания myTask1, myTask1
3. Задания встроены в процесс построения
3.1. myTask1 выполняется перед test
3.2. myTask2 выполняется перед clean
4. логгирование test переопределено
